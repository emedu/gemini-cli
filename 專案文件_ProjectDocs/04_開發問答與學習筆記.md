# 04 開發問答與學習筆記

以 Q&A 格式整理開發過程中的「關鍵問題」與「解決方案」。

## Q1: 如何關閉佔用大量 CPU 與記憶體的 M365Copilot 與 msedgewebview2？

**M365Copilot (Microsoft 365 Copilot)**
* **暫時關閉 (立即釋放效能)**：可在「工作管理員」中找到該處理程序並點擊「結束工作」。
* **長期關閉 (避免開機自啟)**：
  1. 打開 Windows「設定」>「應用程式」>「啟動」。
  2. 找到 Microsoft Copilot 或相關的 M365 應用程式，將其開關設為「關閉」。
  3. 若為企業版，可能需要透過群組原則 (Group Policy) 停用 Windows Copilot。

**msedgewebview2 (Edge WebView2)**
* **注意**：WebView2 是 Windows 與許多應用程式 (如 Teams、新版 Outlook、Widgets 小工具) 用來顯示網頁內容的核心組件。**不建議直接強制解除安裝**，否則會導致依賴它的程式崩潰。
* **關閉建議**：
  1. **關閉背景耗電的來源**：通常是某些常駐程式在呼叫它。打開「工作管理員」>「啟動應用程式」，關閉 Teams 或不常用的雲端硬碟常駐程式。
  2. **關閉 Windows 小工具**：在工具列按右鍵 >「工作列設定」> 關閉「小工具 (Widgets)」，這通常能減少大量的 WebView2 背景程序。
